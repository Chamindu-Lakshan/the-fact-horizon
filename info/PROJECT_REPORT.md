# Fact Horizon Creator Package

### Project Report & Documentation

---

## Overview

**Fact Horizon Creator Package** is a professional, browser-based content creation toolkit designed for YouTube creators, documentary makers, and video storytellers. It provides an all-in-one workspace for generating AI-ready prompts, planning content, crafting scripts, and ensuring video quality — all from a single, beautifully designed interface.

The tool runs entirely in the browser with no server, no sign-up, and no internet dependency (after initial load). Simply open the HTML file and start creating.

---

## What It Does

At its core, Fact Horizon Creator Package helps video creators **generate detailed, professional prompts** for AI video generation tools (like Runway, Sora, Kling, etc.). But it goes far beyond simple prompt generation — it's a complete content creation suite.

### Key Capabilities

- **Prompt Generation** — Build rich, cinematic scene descriptions by selecting camera angles, lighting, color grading, environments, music, and more
- **Document Script Writing** — Generate full-length documentary narration scripts with storytelling beats and psychological engagement hooks
- **Scene-by-Scene Storyboarding** — Break down videos into detailed scenes with visuals, audio, transitions, and timing
- **Thumbnail Design** — Get professional thumbnail concepts with Unsplash image references, color psychology, and layout guidance
- **SEO Optimization** — Generate titles, descriptions, tags, hashtags, and keyword strategies for maximum YouTube reach
- **Content Planning** — Get ideas for series, video angles, and engagement strategies
- **Quality Assurance** — Run pre-publish checklists to ensure your content meets professional standards
- **Audio Design** — Select music moods, sound effects, narrator voices, and subtitle styles
- **Psychology & Engagement** — Apply viewer retention techniques, emotional triggers, and persuasion frameworks

## How It Works

The application is organized into **15 sections**, each accessible from a sidebar navigation menu. Users fill in details using chip selectors, dropdowns, and text fields — then the tool assembles everything into a polished, copy-ready prompt or script.

### Sections at a Glance

| # | Section | Purpose |
|---|---------|---------|
| 1 | **Dashboard** | Home screen with project overview and quick-start options |
| 2 | **Prompt Builder** | Main workspace — assemble full video prompts from 12+ sub-sections |
| 3 | **Camera & Composition** | Choose camera angles, lens types, shot sizes, and depth of field |
| 4 | **Lighting Library** | Select lighting setups, moods, and techniques |
| 5 | **Color Grading** | Pick color palettes, tones, and visual styles |
| 6 | **Environment & Setting** | Define locations, weather, time of day, and atmosphere |
| 7 | **Audio Library** | Music mood, genre, tempo, and sound effects |
| 8 | **Narrator & Subtitles** | Voice style, accent, pacing, subtitle format |
| 9 | **Script Writer** | Generate full documentary narration with storytelling structure |
| 10 | **Scene Storyboard** | Break video into scenes with detailed per-scene prompts |
| 11 | **Psychology & Engagement** | Viewer hooks, retention techniques, emotional triggers |
| 12 | **Thumbnail Generator** | Design eye-catching thumbnail concepts |
| 13 | **SEO & Title Kit** | YouTube titles, descriptions, tags, hashtags |
| 14 | **Content Planner** | Series ideas, video concepts, engagement strategies |
| 15 | **Quality Checklist** | Pre-publish review with scoring system |

## Workflow Examples

### 1. Quick Prompt (2 minutes)
Select a topic → Pick camera + lighting + environment → Click "Generate" → Copy the prompt.

### 2. Full Documentary (15 minutes)
Fill in topic and sub-topics → Build the script → Add storytelling beats → Generate scene-by-scene storyboard → Export everything.

### 3. YouTube Upload Package (10 minutes)
Generate prompt → Create thumbnail → Get SEO titles, tags, and description → Run quality checklist → Upload.

## Design & User Experience

- **Dark cinematic theme** inspired by professional video editing software
- **Card-based layout** for easy scanning and selection
- **Chip selectors** for quick multi-option picking
- **One-click copy** on all generated content
- **Built-in help system** with 5 tabs: Quick Start, Section Guide, Workflows, Shortcuts, and Pro Tips
- **Responsive design** that works on desktop and tablet
- **No dependencies** — pure HTML, CSS, and JavaScript

## Getting Started

1. Open `index.html` in any modern web browser
2. Click **❓ Help** in the top-right corner or press **F1** for the built-in guide
3. Start with the **Dashboard** to understand the workspace
4. Use the **Prompt Builder** to create your first prompt
5. Explore other sections as needed for your content
